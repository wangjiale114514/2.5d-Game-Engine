# 一个2.5d引擎

一个使用纯 C 语言编写的2.5d射线投射引擎，实现了类似经典游戏《doom》的第一人称视角渲染效果。

## 特性

- **射线投射渲染** - 使用射线投射技术实现伪 3D 渲染
- **多扇区支持** - 支持多个独立的空间扇区，可设置不同的地板和天花板高度
- **多扇区支持** - 支持不同高度的渲染
- **传送门系统** - 扇区之间可通过传送门连接，实现无缝场景切换
- **四叉树空间索引** - 使用四叉树优化空间查询，提升性能
- **迷雾效果** - 基于距离的深度雾效果
- **鱼眼矫正** - 修正边缘射线的视角畸变
- **碰撞检测** - 玩家与墙壁的碰撞检测

## 编译与运行

### 环境要求
- Windows 系统
- GCC 编译器
- Windows GDI32 图形库

### 编译
使用提供的批处理脚本编译
```batch
build.bat
```

或手动编译：
```batch
gcc -o game.exe main.c quadtree.c -lgdi32
```

### 运行
```batch
game.exe
```

WASD 控制移动，鼠标控制视角。

关闭窗口可退出程序。

## 技术实现

### 射线投射 (Raycasting)
- 从玩家位置向屏幕每条像素列发射一条射线
- 计算射线与墙壁的距离
- 根据距离计算墙壁在屏幕上的高度
- 应用距离衰减颜色模拟迷雾效果

### 四叉树优化
- 使用四叉树对地图扇区进行空间索引
- 快速查询玩家和射线所在的扇区
- 减少射线投射时的遍历开销

### 传送门渲染
- 支持多层传送门嵌套
- 正确处理遮挡关系
- 根据扇区高度差异渲染地面和天花板

## 地图系统

地图由多个扇区 (Sector) 组成，每个扇区包含：
- 位置坐标 (x, y, w, h)
- 地板高度 (floor)
- 天花板高度 (ceil)
- 六个面的纹理编号（预留）

在 `map.c` 中可以自定义地图布局和各扇区的高度设置。

## 参数调整

### 渲染参数
- `FOV` - 视野角度（默认 60 度）
- `player_HEIGHT` - 玩家眼睛高度
- `WIDTH` / `HEIGHT` - 屏幕分辨率（默认 640x400）

### 玩家参数
- `MOVE_SPEED` - 移动速度（默认 3.0 单位/秒）
- `TURN_SPEED` - 旋转速度（默认 90 度/秒）
- `PLAYER_RADIUS` - 碰撞检测半径

### 四叉树参数
- `MAX_MAPS_PER_NODE` - 每节点最大扇区数
- `MAX_DEPTH` - 最大递归深度
